Which one should I choose?

  • Choose Community Package Setup if you want the broader community setup path.
  • Choose Launcher Setup & Update if you already have Battlefield 1942 configured and only need the launcher.
  • Choose Launcher Setup & Update if you are updating to a newer launcher release.

What happens after install?

  • Open the BF1942 Launcher from the desktop or Start Menu.
  • Let the server list refresh.
  • Double click a server to join.
  • Use the launcher for future updates and multiplayer access.
